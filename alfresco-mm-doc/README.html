Generating Documentation

The Media Management module doc project uses DITA Open Toolkit to generate various outut formats from DITA source.

Generate the XHTML documentation by running the build script from the doc directory: 

$ ./build.sh
You can also specify output to pdf or eclipsehelp, i.e.: 
$ ./build.sh pdf

Viewing Documentation

Once the XHTML has been generated you should be able to view the Documentation.
